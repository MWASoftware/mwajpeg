procedure SaveStretchedPictureToFile(Picture: TPicture; width, height: integer; const FileName: string);

Class: TJPEGFileCompressor

Provided that the Picture holds a Bitmap or a Metafile image, it will be stretched to the specified width and height, and then compressed to a JPEG and written out to the file "FileName".

Note that the image processing is carried out at the current screen resolution. In order to get the best results, you should set your display adapter to the best resolution and colour depth supported by your display adapter/monitor.
