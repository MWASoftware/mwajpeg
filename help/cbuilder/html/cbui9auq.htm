void __fastcall WriteMarker(int marker, const void *buf, int Count);

Class: TJPEGFileCompressor

This method may be called from the event "OnWriteMarkers" to write out data with a JPEG APPn marker. The actual marker is identified by the Marker parameter (e.g. to write out an APP1 marker, set Marker to one). The value data written out is contained in buf and contains Count bytes.
