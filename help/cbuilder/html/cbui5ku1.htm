__property bool AutoDisplay = {read=FAutoDisplay, write=FAutoDisplay, default=1};

Class : TDBJPEGImage

If AutoDisplay is True (the default value), the image automatically displays new data when the underlying BLOB field changes (such as when moving to a new record). If AutoDisplay is False, the image clears whenever the underlying BLOB field changes. To display the data, the user can double-click on the control or select it and press Enter. In addition, calling the LoadPicture method ensures that the control is showing data. Change the value of AutoDisplay to False if the automatic loading of BLOB fields seems to take too long.
