void __fastcall ReadDIBitmapFromStream(Classes::TStream* Stream, tagBITMAPINFO &jpeg_, TJPEGOutputType OutputType, int & bits);

Class: TJPEGFileDecompressor

The source stream is assumed to contain a JPEG image in the JFIF format. This is decoded into a windows device independent bitmap and returned in the memory block given by bits. The output type is determined as above and the information describing the bitmap size, resolution and colour map is returned in the Windows APS BitMapInfo structure.
