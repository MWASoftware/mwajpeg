virtual void __fastcall Abort(void);

Classes: TJPEGFileDecompressor and TJPEGFileCompressor

The Abort method may be called to abandon a compression or decompression operation. It must be called from the same thread as the compression or decompression operation. Practically, this limits its use to an event handler (e.g. OnProgressReport), when it may be used to abort the operation, perhaps in response to a user cancellation request.
