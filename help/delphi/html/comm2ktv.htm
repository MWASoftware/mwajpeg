Units

The MWA JPEG Component Library consists of five units:

MWAJPEG This unit encapsulates the JPEG Compression/Decompression library and defines the two main Delphi Components exported by the library.
JPEGLIB This is the interface unit to the JPEG Compression/Decompression library
JPEGREG1/
JPEGREG2		 This unit contains the Register procedure used to install the components. It may be modified in order to change the palette on which the components are installed. By default, this is the “Additional” palette.

JPEGREG1 is used with Quick Reports 1 (and when Quck Reports is not installed) and JPEGREG2 with Quick Report 2.
MWADBJPG This unit defines the TDBJPEGImage control.
MWAQRJPG This unit defines the TQRDBJPEGImage control.
