TQRDBJPEGImage

TQRDBJPEGImage is used to print images in JPEG format stored in binary (BLOB) fields on a Quick Reports 2 form. It is not used with Quick Reports 1. By default, Quick Reports 2 only comes with Delphi 3 and C++Builder 3. Delphi 2 and C++Builder 1 come with Quick Reports 1 and you will only have Quick Reports 2 with these products (and Delphi 1) if you are registered for the Professional Edition.

Like any Quick Reports Printable control, a TQRDBJPEGImage is placed on a report band to print a JPEG image held in a database field. The DataSet property is set to the dataset containing the image and the DataField property is set to the name of the field holding the JPEG Image.

If non-default decompression parameters are required, then the JPEGDecompressor should point to a TJPEGFileDecompressor object with the required parameter settings.

Note: for printing JPEG Images using Quick Reports 1, you should use the TDBJPEGImage control instead of TQRDBJPEGImage.
