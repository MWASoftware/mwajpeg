procedure SaveDIBitmapToStream(Stream: TStream; const BitmapInfo: TBitmapInfo;bits: PChar);

Class: TJPEGFileCompressor

The device independent bitmap held in the memory block given by "bits", is compressed to a JPEG and written out to the provided stream. Note that the BitmapInfo structure is defined by the Windows API and defines the size and resolution of the bitmap and may also contain a colour map.
