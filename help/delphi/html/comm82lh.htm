TDBJPEGImage Class


Properties Methods Classes

The TDBJPEGImage control displays a graphic image stored in JPEG format from a BLOB (binary large object) field of the current record of a dataset.

Unit: mwadbjpg

Description

Use TDBJPEGImage to represent the value of JPEG graphic fields. TDBJPEGImage allows a form to display graphical data from a dataset. TDBJPEGImage can allow the user to edit graphical data using the clipboard. While the application is running and the database image control has focus, users can cut, copy, and paste using the standard windows accelerator keys (Ctrl+X, Ctrl+C, and Ctrl+V). Users can return to the original state of the database image control by pressing Esc before moving to another record.
