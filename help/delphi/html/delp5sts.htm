function CropBitmap(bitmap: TBitmap; width, height: integer; Clip: TRect): TBitmap;

CropBitmap is a utility function that is used to copy a portion of an existing bitmap (given by the rectangle "Clip" to a new bitmap returned as the result of the function. The newly created bitmap can also be resized. "width and "height" specify the width and height of the new bitmap. If the bitmap is to be the same size as the Clip, then widh should be set to Clip.Right-Clip.Left and height to Clip.Bottom-Clip.Top.
